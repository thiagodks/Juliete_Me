## Juliete_me - Thiago Silva
Juliete_me é um Sotware Inútil, desenvolvido em python, cujo objetivo é basicamente colocar uma juju na foto passada para o programa.

### Requísitos
OpenCv, Colored, PIL

### Execução
```sh
    $ python3 Juliete_me.py -f <foto.*> -o <oculos.* (opcional)>
```
